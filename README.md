## 项目中用到的 ng 指令

### ng-model指令
ng-model 指令绑定了HTML表单中的元素到scope变量中

### ng-minlength,ng-maxlength指令
- ng-minlength: 最小长度
- ng-maxlength: 最大长度

### ng-submit指令
- 该指令用于表单提交后需要执行的指定函数

### ng-class指令
- 该指令可以绑定一个或多个类，并且可以通过函数的返回值 true flase判断是否需要引用该类名

### ng-if指令
- ng-if 指令用于在表达式为 false 时移除 HTML 元素。
如果 if 语句执行的结果为 true，会添加移除元素，并显示。

### ng-disabled指令
- 返回值为 true 禁用表单内容
